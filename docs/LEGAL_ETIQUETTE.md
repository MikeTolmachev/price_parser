
# Legal & Etiquette

This project monitors public web pages.

## Be respectful
- Follow each site’s Terms of Service.
- Respect robots.txt where applicable.
- Use rate limiting (default 1 request / 3–5 seconds).
- Identify with a sane User-Agent.

## Avoid aggressive scraping
- No parallel hammering.
- Back off on HTTP 429/503.

If a site provides an official API, prefer it.
